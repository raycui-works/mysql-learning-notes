2025-12-29.md的文件里，最下边的SQL，如果所有演员都演过1部以上的电影，则表结构只会显示2+，0和1不会显示在表结构上，如果希望哪怕没有数据，也要把0和1显示出来的话，则参考下方SQL。

```sql
SELECT b.film_bucket,
       COALESCE(x.actor_count, 0) AS actor_count
FROM (
  SELECT '0' AS film_bucket, 0 AS sort
  UNION ALL SELECT '1', 1
  UNION ALL SELECT '2+', 2
) b
LEFT JOIN (
  SELECT
    CASE
      WHEN film_count = 0 THEN '0'
      WHEN film_count = 1 THEN '1'
      ELSE '2+'
    END AS film_bucket,
    COUNT(*) AS actor_count
  FROM (
    SELECT a.actor_id, COUNT(fa.film_id) AS film_count
    FROM actor a
    LEFT JOIN film_actor fa ON a.actor_id = fa.actor_id
    GROUP BY a.actor_id
  ) t
  GROUP BY film_bucket
) x
ON x.film_bucket = b.film_bucket
ORDER BY b.sort;
```
