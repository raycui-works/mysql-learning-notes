- 演过电影的演员有多少人？
```sql
-- COUNT()必须这么写，函数字母和括号中间不能有空格

SELECT COUNT(*) AS actors_with_films
FROM (
      SELECT a.actor_id
      FROM actor a
      LEFT JOIN film_actor fa
           ON a.actor_id = fa.actor_id
      GROUP BY a.actor_id
      HAVING COUNT(fa.film_id) > 0
     ) t;```

- 没演过电影的演员有多少人？
```sql
SELECT COUNT(*) AS actors_without_films
FROM (
      SELECT a.actor_id
      FROM actor a
      LEFT JOIN film_actor fa 
           ON a.actor_id = fa.actor_id
      GROUP BY a.actor_id
      HAVING COUNT(fa.film_id) = 0
     ) t;```
